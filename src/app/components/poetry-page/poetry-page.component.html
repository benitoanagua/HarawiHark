about harawihark enhanced

harawihark uses advanced phonetic analysis powered by RiTa.js to provide comprehensive poetic analysis. New features include:

  • • Click words to see intelligent replacement suggestions
  • • Automatic alliteration detection
  • • Part-of-speech tagging for each word
  • • Stress pattern analysis
  • • Cross-line pattern detection
  • • Rhyme and sound-alike word discovery
