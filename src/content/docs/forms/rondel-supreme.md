# Rondel Supreme

**Pattern**: **14 lines of 8 syllables each**.  
Traditional French refrain scheme, but many English poets use the syllabic count only.

- AABBA-ABR-ABBA-ABAR (with refrains)
- Focuses on syllables here, not rhyme.

### Example (first 3 lines)

```
The sky unrolls a velvet sheet
Streetlights blink like tired eyes
A cat slips through the lullabies
```
