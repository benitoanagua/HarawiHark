# Lanterne

**Pattern**: 1-2-3-4-1 syllables over **5 lines**, centered to resemble a lantern.

- Japanese-inspired form shaped like a paper lantern.
- Creates visual symmetry with the syllable pattern.
- Often centered on the page for visual effect.

### Example

```
    Light
  glows soft
 in the darkness
illuminating dreams
    peace
```

### Spanish Example

```
    Luz
  brilla
 en la noche
iluminando sueños
    paz
```
