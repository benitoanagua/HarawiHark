# Cinquain (Adelaide Crapsey)

**Pattern**: 2-4-6-8-2 syllables over **5 lines**.

- American Imagist twist on traditional forms.
- Shape resembles a gentle crescendo and diminuendo.
- Created by poet Adelaide Crapsey (1878-1914).

### Example

```
Night
windswept
darkness hums
a silent chorus
hush
```

### Spanish Example

```
Luz
dorada
el sol desciende
entre nubes de colores
paz
```
