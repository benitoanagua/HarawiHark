# Fib (Fibonacci Poem)

**Pattern**: 1-1-2-3-5-8 syllables over **6 lines**.

- Inspired by Fibonacci sequence; coined by Gregory K. Pincus.
- Each line follows the mathematical pattern: 1, 1, 2, 3, 5, 8...
- Creates natural crescendo effect.

### Example

```
I
am
watching
raindrops race
down the window slowly
each one a tiny universe complete
```

### Spanish Example

```
Yo
veo
estrellas
brillan arriba
en la noche clara y fría
susurran secretos antiguos del cosmos
```
