# My line count is wrong – why?

1. **Accents / diacritics** – make sure they are UTF-8.
2. **Hyphenated words** – each part is counted.
3. **Punctuation** – ignored, but em-dashes and en-dashes can create false syllables.
4. **Loanwords** – e.g., “naïve” may count 2 or 3 depending on rule set.

If still stuck, open an issue with the exact text.
