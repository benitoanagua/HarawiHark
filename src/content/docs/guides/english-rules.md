# English Syllable Counting Rules

HarawiHark uses the advanced **simi-syllable** library for accurate English syllable counting. The algorithm handles:

- **Diphthongs and triphthongs** (e.g., "beautiful" → 3 syllables)
- **Silent "e"** at word endings
- **Suffix variations** (-ed, -es, -ing)
- **Compound words** and common exceptions
- **Advanced phonetic patterns** beyond simple vowel counting

### Example Accuracy

```
"computer" → com-pu-ter → 3 syllables ✓
"beautiful" → beau-ti-ful → 3 syllables ✓
"interesting" → in-ter-est-ing → 4 syllables ✓
```
