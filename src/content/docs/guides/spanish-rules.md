# Spanish Syllable Count – Edge Cases

| Rule                   | Example                        | Count |
| ---------------------- | ------------------------------ | ----- |
| **Sinalefa**           | _al amanecer_ → a-la-ma-ne-cer | 5     |
| **Diéresis**           | _país_ → pa-ís                 | 2     |
| **Hiato**              | _león_ → le-ón                 | 2     |
| **Diéresis explícita** | _pingüino_ → pin-güi-no        | 3     |

We use `hyphen-es` + custom rules to handle these automatically.
