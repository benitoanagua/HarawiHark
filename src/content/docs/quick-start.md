# HarawiHark – 3-Minute Quick-Start

HarawiHark is a tiny web-tool that **checks if your poem follows a chosen syllable pattern**.  
Paste your lines, pick a form, get instant feedback.

## 1-Minute Demo

1. Open [harawi-hark.vercel.app](https://harawi-hark.vercel.app)
2. Select **Haiku** (5-7-5).
3. Type:

```
An old silent pond
A frog jumps into the pond—
Splash! Silence again.
```

4. ✅ All three lines match 5-7-5.

## Supported Languages

- **Spanish** – handles _sinalefa_, _diéresis_, _hiatos_
- **English** – counts schwa reductions correctly
