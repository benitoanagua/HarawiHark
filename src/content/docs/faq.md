# FAQ

**Q: Can I add my own form?**  
A: Yes – add a JSON file in `src/lib/forms` and restart dev.

**Q: Does HarawiHark store my poems?**  
A: No – everything runs in the browser or is prerendered; nothing is sent to a server.

**Q: Offline support?**  
A: Install as PWA; the service worker caches all routes.

**Q: Browser support?**  
A: Modern Chromium, Firefox, Safari. IE11 is not supported.
