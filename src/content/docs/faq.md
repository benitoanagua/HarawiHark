# FAQ

**Q: How many poetry forms does HarawiHark support?**  
A: Currently **8 essential forms**: haiku, tanka, cinquain, limerick, redondilla, lanterne, diamante, and fibonacci. We focus on the most useful and accessible patterns.

**Q: Can I add my own form?**  
A: The current version supports our curated selection of 8 forms. For custom patterns, you can use our API programmatically or request new forms via GitHub issues.

**Q: Does HarawiHark store my poems?**  
A: No – everything runs in your browser. No poems are sent to servers or stored anywhere.

**Q: Why these specific 8 forms?**  
A: We chose forms that are: widely recognized, teachable, culturally significant, and have clear syllabic patterns. This covers Japanese (haiku, tanka), Spanish (redondilla), English (limerick, cinquain), and creative modern forms.

**Q: What about rhyme schemes?**  
A: HarawiHark focuses only on syllable counting, not rhymes. Many of these forms traditionally have rhyme patterns, but we validate just the metric structure.

**Q: Offline support?**  
A: Yes! Install as a PWA and use offline. The syllable counting algorithms work entirely client-side.

**Q: Browser support?**  
A: Modern browsers (Chrome, Firefox, Safari, Edge). No IE11 support.

**Q: Can I contribute new forms?**  
A: Absolutely! Open an issue on GitHub with the form name, pattern, examples, and cultural context. We prioritize forms that are accessible and widely useful.
